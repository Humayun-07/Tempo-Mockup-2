Screen 25 · System Feedback

Toast / Snackbar States

All toast types. Bottom-anchored, 4s auto-dismiss. Each HTTP error code has its own copy. API §5.2.

✕

🔒

AI UNAVAILABLE (HTTP 503)
🤖
AI is taking a short break. Your tasks are saved.
Retry
API §5.2: Every HTTP error code has its own human copy. Rate limit (429) = amber. Auth errors (401, 403) = neutral dark + action. Slot conflict (409) = red. AI down (503) = neutral + retry. Never "Something went wrong."
